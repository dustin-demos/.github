# dustin-demos

A history of things and demos I've built, old and new. Some are years old and kept running close to their original form; others are recent. Mostly small apps and games.

Many of the client-side demos are bundled and served together at **[client-demos.fly.dev](https://client-demos.fly.dev/)**.

- **Python Todo** — an end-to-end typesafe todo app, from a Python FastAPI backend to a TypeScript client via OpenAPI.
- **Reasonable** — a visual node editor for building workflows, with a hand-built drag-and-drop canvas.
- **Clickmart** — a mock storefront UI with departments, a product market, and a carousel.
- **Currency Exchange** — a currency converter built as a job-application code challenge.
- **Onclick Notes** — a serverless markdown notes app that packs the whole note into the URL.
- **Framework Challenge** — the same UI built three ways: Pocket JS, React, and Vue 3.
- **Valorant Blog** — a flat-file CMS for editing a Valorant tips blog.
- **State Sync** — a proof of concept for syncing state across differently-sized frames over a broadcast channel.
- **Resize** — an aspect-ratio calculator for cropping and resizing art.
- **Discord Message Queue** — queues chat messages and drips them into a Discord channel on a timer.
- **Almost Realtime** — a small group chat using HTTP to send messages and SSE to sync them.
- **DVD Video** — the bouncing DVD-logo screensaver, done in pure CSS.
- **h8ball** — a magic eight-ball that will destroy your hopes and dreams.
- **Vue Queue Items** — the same queue UI built three times: v1 emits state up, v2 keeps it local, v3 moves it into Vuex.
- **Roguelike** — a Vue-powered web roguelike built with friends.
- **CSS 3D Toggle Button** — a toggle button with a CSS 3D flip animation.
- **Vue Pagination** — a pagination component in Vue 2.
- **Emoji** — a single emoji, rendered large.
- **SVG Digital Clock** — a digital clock rendered in SVG.
- **iOS Chat Bubbles** — pure CSS iOS-style chat bubbles written as a Sass mixin, one div per bubble.
- **jQuery Dropdown Menu** — a deeply nested dropdown written as a jQuery plugin; the DOM is the state.
- **Messenger** — a recreation of the Messenger UI in pure CSS.

Each demo's full story lives in its repo and on [dustindowell.com](https://dustindowell.com).
